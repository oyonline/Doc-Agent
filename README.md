# 智能体工作台

智能体工作台是一个管理智能体交付的工作台，不是聊天机器人，也不是普通任务管理工具。

它的核心主轴是任务：用户输入目标，系统生成计划，用户确认后智能体执行，系统展示上下文、日志、改动、验证结果，最后生成审查包，由人做最终决策。

## 当前阶段

V0.1.1 整合版：可点击、可走查的低保真交互原型。

当前只验证：

```text
目标 → 计划 → 任务 → 运行 → 验证 → 审查包 → 人工决策
```

不做真实后端、数据库、真实模型调用、真实智能体运行、真实 Git / GitHub 集成、登录、权限、多用户、计费、移动端、多主题、真实工作流库、真实智能体档案、真实设置页或合并队列。
