# 文案统一表

| 统一文案 | 含义 | 出现位置 | 是否需要二次确认 | 备注 |
|---|---|---|---|---|
| 生成任务计划 | 把用户目标转成建议计划和验收标准 | 首页 | 否 | 点击后进入计划确认页 |
| 确认并执行 | 用户确认计划并开始执行 | 计划确认页 | 否 | 进入任务详情页 |
| 保存草稿 | 保存当前计划但不开始执行 | 计划确认页 | 否 | V0.1 可作为静态反馈 |
| 查看运行详情 | 查看一次运行的时间线、上下文、日志、改动和验证 | 任务详情页 | 否 | 进入执行详情页 |
| 查看审查包 | 进入最终决策入口 | 任务板、任务详情页、执行详情页 | 否 | 待审查状态下重点展示 |
| 接受结果 | 接受智能体本次交付 | 审查包页 | 是 | 确认后任务进入已完成 |
| 拒绝并反馈 | 拒绝结果并填写反馈 | 审查包页 | 否 | 必须填写反馈 |
| 重新执行 | 不带新反馈，按原计划重新执行 | 审查包页更多操作、任务详情页 | 是 | 与拒绝并反馈区分 |
| 让审查智能体复查 | 请求审查智能体再检查一次 | 审查包页更多操作 | 否 | V0.1 可展示静态复查中状态 |
| 停止 | 停止当前可运行任务 | 任务详情页 | 是 | 任务显示已停止标签 |
| 返回任务板 | 返回任务看板 | 审查包页 | 否 | 通常用于决策完成后 |
| 展开差异 | 展示选中文件的静态差异预览 | 审查包页 | 否 | 同页展示，不跳转 |
| 更多操作 | 收纳次要操作 | 审查包页 | 否 | 包含复查和重新执行 |
| 保存为工作流 | 将成功经验沉淀为工作流 | 审查包页决策后提示 | 否 | V0.1 可作为后续入口提示 |
| 暂不保存 | 不把本次经验保存为工作流 | 审查包页决策后提示 | 否 | V0.1 可作为静态按钮 |
