## 动作名称统一

| 统一文案 | 使用位置 | 含义 | 不使用的说法 |
|---|---|---|---|
| 生成任务计划 | 首页 | 把用户目标转成建议计划和验收标准 | 生成计划 / 创建计划 |
| 确认并执行 | 计划确认页 | 用户确认计划并开始执行 | 开始执行 / 执行 |
| 查看运行详情 | 任务详情页、历史运行列表 | 进入执行详情页 | 查看日志 / 查看改动作为顶部按钮 |
| 查看审查包 | 任务板、任务详情页、执行详情页 | 进入最终决策入口 | 进入审查 |
| 接受结果 | 审查包页 | 用户认可本次结果 | 通过 / 合并 |
| 拒绝并反馈 | 审查包页 | 用户拒绝本次结果并提交反馈 | 拒绝 / 驳回 |
| 重新执行 | 审查包页更多操作 | 不带新反馈，按原计划重新跑一次 | 重跑 / 再执行一次 |
| 让审查智能体复查 | 审查包页更多操作 | 让审查智能体再次检查 | 复查 / 智能复核 |
| 停止 | 任务详情页 | 停止当前运行 | 中止 / 终止 |
| 重试 | 任务详情页 | 从失败、停止或卡住处重新发起一次运行 | 重新执行 |
| 返回首页 | 占位页 | 回到首页 | 返回 |
| 返回任务板 | 执行详情页、审查包页 | 回到任务板 | 返回列表 / 返回任务 |
| 暂不保存 | 接受结果后的工作流提示 | 不把本次经验保存为工作流 | 以后再说 |

## 状态文案统一

| 场景 | 统一文案 |
|---|---|
| 待审查提示 | 等待你审查 |
| 人工处理提示 | 需要人工处理 |
| 运行时长 | 已运行 X 分 X 秒 |
| 验证通过 | 测试全部通过 |
| 验证未通过 | 验证未通过 |
| 执行失败 | 执行失败 |
| 停止完成 | 当前运行已停止 |
| 拒绝反馈提交 | 反馈已进入下一次运行的上下文 |

## 术语统一

- 智能体，不用 Agent / 代理。
- 工作流，不用 Workflow。
- 审查包，不用 Review Packet。
- 接受结果，不用通过 / 合并。
- 拒绝并反馈，不用拒绝 / 驳回。
- 重新执行，不用重跑。
- 停止，不用中止 / 终止。
