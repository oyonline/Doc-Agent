# 评审意见处理记录

## 处理原则

本记录用于归档 `评审意见.md` 中的问题，并说明哪些已被 V0.1.1 整合版吸收，哪些不进入当前版本。

状态只能使用：

- 已解决。
- 部分解决。
- 暂不进入 V0.1.1。
- 待确认。

当前处理基准：

1. 最高事实源：`docs/01-单一事实源-V0.1.1.md`。
2. 评审意见若已被 V0.1.1 吸收，不再列为开放问题。
3. V0.2 或后续能力只作为锚点，不写成 V0.1.1 必做。
