# 页面跳转矩阵

本文件为旧跳转矩阵入口。

当前完整跳转矩阵和按钮状态变化见：

- `docs/03-状态机与跳转矩阵.md`

## 当前核心跳转

| 来源页面 | 触发动作 | 目标页面 | 状态变化 |
|---|---|---|---|
| 首页 | `生成任务计划` | 计划确认页 | 暂不创建执行运行 |
| 计划确认页 | `确认并执行` | 任务详情页 | 待处理 → 执行中 |
| 任务板 | 任务卡片 | 任务详情页 | 不变化 |
| 任务板 | `查看审查包` | 审查包页 | 不变化 |
| 任务详情页 | `查看运行详情` | 执行详情页 | 不变化 |
| 任务详情页 | `查看审查包` | 审查包页 | 不变化 |
| 执行详情页 | `查看审查包` | 审查包页 | 验证通过后可进入；任务保持待审查 |
| 审查包页 | 修改文件清单行 | 审查包页同页差异预览 | 不变化 |
| 审查包页 | `接受结果` | 任务板 | 待审查 → 已完成 |
| 接受结果提示 | `保存为工作流` | 工作流占位页 | 已完成，展示已保存工作流示意卡 |
| 接受结果提示 | `暂不保存` | 任务板 | 已完成，未保存为工作流 |
| 审查包页 | `拒绝并反馈` | 任务详情页 | 待审查 → 执行中 + 已拒绝 |
| 审查包页 | `重新执行` | 任务详情页 | 待审查 → 执行中 |
| 审查包页 | `让审查智能体复查` | 审查包页 | 主状态保持待审查 |
| 工作流占位页 | `返回首页` | 首页 | 不变化 |
| 智能体占位页 | `返回首页` | 首页 | 不变化 |
| 设置占位页 | `返回首页` | 首页 | 不变化 |

## 废弃跳转

以下跳转已经废弃：

- 审查包页修改文件清单 → 执行详情页。
- 左侧导航“运行” → 运行列表页。
- 左侧导航“审查” → 审查队列页。
