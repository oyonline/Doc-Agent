# 智能体工作台 V0.1.1 低保真原型

本目录是 V0.1.1 可点击、可走查的低保真静态原型。

入口文件：

- `index.html`

打开方式：

直接用浏览器打开 `prototype/low-fi/index.html`。

## 原型范围

本原型只验证主链路：

目标 → 计划 → 任务 → 运行 → 验证 → 审查包 → 人工决策 → 工作流沉淀。

包含页面：

- 首页。
- 计划确认页。
- 任务板。
- 任务详情页。
- 执行详情页。
- 审查包页。
- 工作流、智能体、设置三个占位页。

不包含真实后端、数据库、模型调用、真实智能体执行、真实工作流管理或依赖安装。

## 主要交互

- 首页点击 `生成任务计划` 进入计划确认页。
- 计划确认页点击 `确认并执行` 进入任务详情页。
- 任务详情页点击 `查看运行详情` 进入执行详情页。
- 执行详情页点击 `查看审查包` 进入审查包页。
- 审查包页支持 `接受结果`、`拒绝并反馈`、`重新执行`、`让审查智能体复查`。
- 接受结果后可选择 `保存为工作流` 或 `暂不保存`。
- 工作流页在保存后展示“前端问题修复”静态示意卡片。

## 样例目标

修复登录页错误提示不显示，并补充测试。

## 说明

本原型使用原生 HTML、内联 CSS 和少量原生 JavaScript。
所有状态变化都是静态原型展示，不代表真实执行。
