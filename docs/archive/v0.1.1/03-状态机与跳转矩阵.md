# 状态机与跳转矩阵

## 文档目的

本文件集中整理 V0.1.1 的状态、异常标签、页面跳转和按钮触发后的状态变化，避免设计师和前端对同一交互产生不同理解。

当前唯一事实源仍是 `docs/01-单一事实源-V0.1.1.md`。本文件是它的执行矩阵。

## 主任务状态

主任务状态只保留五个：

```text
待处理 → 执行中 → 验证中 → 待审查 → 已完成
```

| 主状态 | 含义 | 看板列 |
|---|---|---|
| 待处理 | 已创建但未开始，或保存草稿后等待确认执行 | 是 |
| 执行中 | 工作智能体正在执行，或任务处于执行阶段 | 是 |
| 验证中 | 正在验证，或停在验证阶段等待处理 | 是 |
| 待审查 | 审查包已生成，等待人工决策 | 是 |
| 已完成 | 用户已接受结果 | 是 |

## 异常标签

异常标签不作为主状态，也不作为任务板列名。

| 异常标签 | 使用场景 | 后续入口 |
|---|---|---|
| 已停止 | 用户停止当前运行 | 任务详情页 `重试` |
| 执行失败 | 执行阶段失败 | 任务详情页 `重试`，执行详情页查看日志 |
| 验证未通过 | 验证阶段未通过 | 任务详情页 `重试`，执行详情页查看验证 |
| 已拒绝 | 用户在审查包页提交了拒绝反馈 | 新一次运行上下文展示反馈 |
| 卡住 | 长时间没有推进的静态提示 | 任务详情页 `重试` 或人工查看 |
| 需要人工处理 | 需要用户介入判断 | 任务详情页、执行详情页、审查包页 |

## 运行状态

运行是单次执行记录，出现在任务详情页“历史运行”和执行详情页，不作为一级导航。

| 运行状态 | 含义 |
|---|---|
| 未开始 | 运行记录已准备但尚未启动 |
| 进行中 | 当前运行正在执行 |
| 已完成 | 当前运行完成，结果可进入验证或审查 |
| 已失败 | 当前运行失败 |
| 已停止 | 当前运行被用户停止 |

## 验证状态

| 验证状态 | 含义 |
|---|---|
| 未开始 | 尚未进入验证 |
| 进行中 | 正在运行验证 |
| 已通过 | 所有验证项通过 |
| 未通过 | 至少一个验证项未通过 |

验证状态出现在任务卡、任务详情页验收标准、执行详情页验证 Tab、审查包验证结果区。

## 智能体状态

右侧状态栏使用三类智能体状态：

| 智能体 | 状态 |
|---|---|
| 计划智能体 | 空闲 / 工作中 |
| 工作智能体 | 空闲 / 工作中 |
| 审查智能体 | 空闲 / 工作中 |

样例：

- 计划智能体：空闲。
- 工作智能体：工作中 · 任务-001 · 已运行 4 分 12 秒。
- 审查智能体：空闲。

## 关键异常回路

```text
待处理
  │ 确认并执行
  ▼
执行中
  │ 执行完成
  ▼
验证中
  │ 验证通过
  ▼
待审查
  │ 接受结果
  ▼
已完成

执行中
  │ 执行失败
  ▼
执行中 + 执行失败 + 需要人工处理
  │ 重试
  ▼
执行中

验证中
  │ 验证未通过
  ▼
验证中 + 验证未通过 + 需要人工处理
  │ 重试
  ▼
执行中

执行中 / 验证中
  │ 停止
  ▼
原主状态 + 已停止
  │ 重试
  ▼
执行中

待审查
  │ 拒绝并反馈
  ▼
执行中 + 已拒绝

待审查
  │ 重新执行
  ▼
执行中
```

## 页面跳转矩阵

| 来源页面 | 触发动作 | 目标页面 | 状态变化 | 备注 |
|---|---|---|---|---|
| 顶栏 | `创建目标` | 首页 | 不改变状态 | 聚焦目标输入框 |
| 首页 | `生成任务计划` | 计划确认页 | 暂不创建执行运行 | 展示原始目标、6 步计划、3 条验收标准 |
| 首页 | `只创建任务` | 首页 | 新增待处理任务卡 | 任务名默认取目标前 20 个字 |
| 首页 | 当前任务卡片 | 任务详情页 | 不改变状态 | 快速回到任务 |
| 首页 | 待审查提示条 | 任务板 | 不改变状态 | 定位到待审查列 |
| 计划确认页 | `确认并执行` | 任务详情页 | 待处理 → 执行中 | 开始样例任务执行 |
| 计划确认页 | `保存草稿` | 首页 | 保持待处理 | 可从待处理任务恢复计划确认视图 |
| 计划确认页 | `修改目标` | 首页 | 不改变状态 | 保留输入内容 |
| 任务板 | 任务卡片 | 任务详情页 | 不改变状态 | 查看任务详情 |
| 任务板 | `查看审查包` | 审查包页 | 不改变状态 | 仅待审查任务显示该入口 |
| 任务板空状态 | `返回首页` | 首页 | 不改变状态 | 统一空状态行为 |
| 任务详情页 | `查看运行详情` | 执行详情页 | 不改变状态 | 默认选中最新运行 |
| 任务详情页 | 历史运行行 | 执行详情页 | 不改变状态 | 选择对应运行 |
| 任务详情页 | `停止` | 任务详情页 | 原主状态 + 已停止 | 需要二次确认 |
| 任务详情页 | `重试` | 任务详情页 | 执行失败 / 验证未通过 / 已停止 / 卡住 → 执行中 | 新建一次运行 |
| 任务详情页 | `查看审查包` | 审查包页 | 不改变状态 | 待审查或已完成且已有审查包时出现 |
| 执行详情页 | `返回任务板` | 任务板 | 不改变状态 | 返回看板 |
| 执行详情页 | `查看审查包` | 审查包页 | 验证通过后可进入；任务保持待审查 | 验证未通过时置灰 |
| 执行详情页 | 任务名 / 返回任务详情 | 任务详情页 | 不改变状态 | 回看任务上下文 |
| 审查包页 | 修改文件清单行 | 审查包页同页差异预览 | 不改变状态 | 不跳转，不离开审查包页 |
| 审查包页 | `接受结果` | 任务板 | 待审查 → 已完成 | 弹出“是否保存为工作流？”，选择 `暂不保存` 后返回 |
| 接受结果提示 | `保存为工作流` | 工作流占位页 | 已完成，展示已保存工作流示意卡 | 只展示静态保存结果，不开放真实工作流管理 |
| 接受结果提示 | `暂不保存` | 任务板 | 已完成，未保存为工作流 | 任务进入已完成列 |
| 审查包页 | `拒绝并反馈` | 任务详情页 | 待审查 → 执行中 + 已拒绝 | 必须填写反馈，反馈进入下一次运行上下文 |
| 审查包页 | `重新执行` | 任务详情页 | 待审查 → 执行中 | 不带新反馈，新建一次运行 |
| 审查包页 | `让审查智能体复查` | 审查包页 | 主状态保持待审查 | 右侧状态栏显示审查智能体工作中 |
| 审查包页 | `返回任务板` | 任务板 | 不改变状态 | 定位任务所在列 |
| 工作流占位页 | `返回首页` | 首页 | 不改变状态 | 占位页统一行为 |
| 智能体占位页 | `返回首页` | 首页 | 不改变状态 | 占位页统一行为 |
| 设置占位页 | `返回首页` | 首页 | 不改变状态 | 占位页统一行为 |

## 每个按钮触发后的状态变化

| 按钮 | 所在页面 | 是否需要二次确认 | 是否需要反馈 | 状态变化 | 说明 |
|---|---|---|---|---|---|
| `生成任务计划` | 首页 | 否 | 否 | 不进入执行状态 | 进入计划确认页 |
| `只创建任务` | 首页 | 否 | 否 | 新增待处理任务 | 不进入执行 |
| `确认并执行` | 计划确认页 | 否 | 否 | 待处理 → 执行中 | 开始第一次运行 |
| `保存草稿` | 计划确认页 | 否 | 否 | 保持待处理 | 后续可从待处理任务恢复 |
| `查看运行详情` | 任务详情页 | 否 | 否 | 不变化 | 进入执行详情页 |
| `停止` | 任务详情页 | 是 | 否 | 原主状态 + 已停止 | 当前运行变为已停止 |
| `重试` | 任务详情页 | 否 | 否 | 执行失败 / 验证未通过 / 已停止 / 卡住 → 执行中 | 新建一次运行，不等同于审查包页的重新执行 |
| `查看审查包` | 任务板、任务详情页、执行详情页 | 否 | 否 | 不变化 | 验证未通过时在执行详情页置灰 |
| `接受结果` | 审查包页 | 低风险否，高风险是 | 否 | 待审查 → 已完成 | 弹出是否保存为工作流提示 |
| `保存为工作流` | 接受结果提示 | 否 | 否 | 已完成，展示工作流沉淀状态 | 跳转到工作流占位页并展示静态示意卡 |
| `暂不保存` | 接受结果提示 | 否 | 否 | 已完成，未保存为工作流 | 返回任务板 |
| `拒绝并反馈` | 审查包页 | 否 | 是，必填 | 待审查 → 执行中 + 已拒绝 | 反馈进入下一次运行上下文 |
| `重新执行` | 审查包页更多操作 | 是 | 否 | 待审查 → 执行中 | 按原计划新建一次运行 |
| `让审查智能体复查` | 审查包页更多操作 | 否 | 否 | 主状态保持待审查 | 右侧状态栏显示审查智能体工作中 |
| `返回首页` | 占位页 | 否 | 否 | 不变化 | 回到首页 |
| `返回任务板` | 执行详情页、审查包页 | 否 | 否 | 不变化 | 回到任务板 |

## 验证未通过规则

当验证未通过时：

- 任务主状态停留在“验证中”。
- 叠加“验证未通过”和“需要人工处理”标签。
- 执行详情页 `查看审查包` 置灰。
- 置灰提示固定为：“验证未通过时无法查看审查包，请先重试或人工查看日志。”
- 用户可从任务详情页点击 `重试`，新建一次运行并进入执行中。
