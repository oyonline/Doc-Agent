# 页面跳转矩阵

| 来源页面 | 触发动作 | 目标页面 | 状态变化 | 备注 |
|---|---|---|---|---|
| 首页 | 生成任务计划 | 计划确认页 | 生成待确认计划 | 展示原始目标、建议计划和验收标准 |
| 首页 | 点击当前任务概览卡片 | 任务详情页 | 不变化 | 用于快速回到进行中的任务 |
| 计划确认页 | 确认并执行 | 任务详情页 | 待处理变为执行中 | 开始样例任务执行 |
| 计划确认页 | 保存草稿或返回 | 首页 | 保持待处理 | 不进入执行 |
| 任务板 | 点击任务卡片 | 任务详情页 | 不变化 | 查看任务详情 |
| 任务板 | 查看审查包 | 审查包页 | 不变化 | 仅待审查任务显示该入口 |
| 任务详情页 | 查看运行详情 | 执行详情页 | 不变化 | 查看一次运行证据 |
| 任务详情页 | 查看审查包 | 审查包页 | 不变化 | 待审查状态下可进入 |
| 执行详情页 | 查看审查包 | 审查包页 | 验证中变为待审查 | 验证通过后进入审查 |
| 审查包页 | 接受结果 | 任务板 | 待审查变为已完成 | 需要二次确认 |
| 审查包页 | 拒绝并反馈 | 任务详情页 | 待审查变为执行中，并添加已拒绝标签 | 必须填写反馈，反馈进入下一次运行上下文 |
| 审查包页 | 点击修改文件清单 | 同页差异预览 | 不变化 | 不离开审查包页 |
| 审查包页 | 重新执行 | 任务详情页 | 待审查变为执行中 | 需要二次确认，不带新反馈 |
| 工作流占位页 | 返回首页 | 首页 | 不变化 | 占位页统一行为 |
| 智能体占位页 | 返回首页 | 首页 | 不变化 | 占位页统一行为 |
| 设置占位页 | 返回首页 | 首页 | 不变化 | 占位页统一行为 |
