## 当前项目大目标

V0.2 把"一个需求由多智能体协作交付"作为主轴,围绕:

```text
需求 → 多轮澄清 → PRD → 任务拆解 → 多智能体并行执行 → 单任务审查 → 需求级审查 → 需求接受
```

建立需求交付闭环。

V0.2 是高保真静态原型,所有澄清对话、PRD、拆解结果、任务执行、审查内容都使用静态样例数据。当前阶段先把多智能体协作的人机交互形态走通,再进入真实实现。
